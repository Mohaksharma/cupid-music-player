❓ YOUR SPECIFIC QUESTIONS

Q: "How do I add music?"
A: Copy songs to ~/Music/ folder or read QUICK_BUILD_GUIDE.md

Q: "How do I make a .EXE file?"
A: Run: npm run build && npm run package
   Then find .exe in out/ folder
   (See QUICK_BUILD_GUIDE.md for details)

Q: "How do I add Spotify?"
A: 1. Create Spotify app
   2. Copy Client ID to .env file
   3. Restart: npm run dev
   4. Click "Connect to Spotify"
   (See QUICK_BUILD_GUIDE.md or BUILD_AND_DISTRIBUTE.md)
